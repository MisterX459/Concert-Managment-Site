Concert managment site created using Express.js and sqlite for database.
3 connected database tables (including one many-to-many relationship with at least one additional column in addition to foreign keys in the connecting table)
Functionalities:
•	Adding, updating, deleting records to the database
•	Displaying a list of all records for each table (only the most important columns)
•	Displaying detailed views (all columns + records connected by relationships)
•	Data validation on the client and server side

•	Frontend and backend application code
•	Database creation script
•	Database script for creating sample data





If there is problems with compilation try this:
npm install
npm install node.js
npm start
